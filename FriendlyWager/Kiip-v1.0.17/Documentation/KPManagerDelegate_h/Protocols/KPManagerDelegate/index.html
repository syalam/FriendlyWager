KPManagerDelegate

Extends Protocol:
NSObject
Declared In:
KPManagerDelegate.h

Introduction

This protocol defines optional method callbacks for asynchronous calls from KPManager as well as notification state callbacks.


Methods

-didCloseNotification:

￼Notifies the delegate when the notification has closed. A notification can close when the banner disappears from the view position or the webview has been dismissed.

-didCloseWebView:

Notifies the delegate when the webview has been closed.

-didPresentNotification:

￼Notifies the delegate when the notification has been shown on the screen.

-didReceiveContent:quantity:

Notifies the delegate that the user has won some in-game content.

-didReceiveContent:quantity:withReceipt:

Notifies the delegate that the user has won some in-game content.

-didShowWebView:
